# Unique Greek Words from a text.

A programm for getting a list with unique Greek Words from a text.

## How to use it

You need the _./dist_ folder.

-   import class StringWorker from _./classes/StringWorker.js_ <br />
    `import { StringWorker } from "./classes/StringWorker.js";`

-   Create an instance of the StringWorker class: <br />
    `const text = new StringWorker("your text in Greek")`

-   Get a list with unique Greek Words from your text using _getList_ method: <br />
    `text.getList()`

***You can find a code example in *main.js* file***
